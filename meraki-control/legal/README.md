# Meraki Legal Documents

Scanned documentation for Meraki Family Limitada corporate compliance.

## Folder Structure

```
legal/
├── contracts/          # Service & rental agreements
│   ├── angelina-equipment-rental/    # Kitchen equipment lease
│   └── earth-movement-services/      # Transport & machinery
├── assemblies/         # Shareholder meeting minutes
│   └── YYYY-MM-DD_asamblea_*.pdf
├── libros/            # Legal books
│   ├── libro-actas/
│   ├── libro-socios/
│   └── libro-contabilidad/
├── permits/           # Operating permits
│   ├── hacienda/
│   ├── municipalidad/
│   ├── ministerio-salud/
│   └── ccss-ins/
├── tax-filings/       # Tax declarations & receipts
│   ├── 2025/
│   └── 2026/
└── corporate/         # General corporate docs
    ├── personeria/
    ├── poderes/
    └── constitutiva/
```

## Naming Convention

- `YYYY-MM-DD_description.pdf`
- Use lowercase, hyphens instead of spaces
- Include entity name if multi-company: `meraki_`, `earth-movement_`, `angelina_`
