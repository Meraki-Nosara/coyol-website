# Meraki Control System

**Unified financial intelligence for Meraki restaurant group**

Owner: Angelina Peri

## Restaurants
1. La Luna
2. Coyol  
3. Esh

## Features

### 📊 Daily Sales Tracking
- Simple daily input (30 seconds)
- Cash + card breakdown
- Customer count
- Revenue by restaurant

### 💰 Unified Accounting
- All income flows to one view
- Daily/weekly/monthly summaries
- Year-over-year comparison
- Payment method analytics

### 📦 Supplier Management
- Track all vendors
- Price comparison across suppliers
- Price history tracking
- Alert on price increases
- Recommend best value options

### 🧠 Seasonal Intelligence
- Historical pattern analysis
- Labor cost recommendations
- Inventory forecasting
- Pre-season alerts
- "This time last year" insights

### 📱 Reports & Alerts
- Weekly digest (WhatsApp/Email)
- Anomaly detection
- Labor % tracking
- Custom recommendations

## Tech Stack
- Frontend: Astro + Tailwind (fast, mobile-friendly)
- Backend: Node.js + SQLite (simple, portable)
- Reports: Automated via cron
- Delivery: WhatsApp/Telegram/Email
